Página de Projetos - Como Adicionar um Banco de Dados em Flask

Bem-vindo à página de projetos! Aqui vamos explicar como adicionar um banco de dados em um projeto Flask.

Passos para Adicionar um Banco de Dados em Flask

  1. Abra o Prompt de Comando (Command Prompt) no Windows.
  2. Crie um ambiente virtual (opcional, mas recomendado) para isolar seu projeto. Você pode usar o seguinte comando:
python -m venv venv
  1. Ative o ambiente virtual com o comando:
venv\Scripts\activate
  1. Use o pip para instalar o Flask e a extensão Flask-SQLAlchemy para integração com o banco de dados:
pip install Flask Flask-SQLAlchemy
  1. Importe e configure o SQLAlchemy em seu aplicativo:

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///meu_banco_de_dados.db'
db = SQLAlchemy(app)
  1. Crie uma classe para representar uma tabela no banco de dados:

class Usuario(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    nome = db.Column(db.String(100), nullable=False)
    email = db.Column(db.String(100), unique=True, nullable=False)
  1. Crie as tabelas no banco de dados usando o seguinte comando:
python
from seu_app import db
db.create_all()

Exemplo de Uso do Banco de Dados

Aqui está um exemplo básico de como criar e consultar registros no banco de dados:


# Criando um novo usuário
novo_usuario = Usuario(nome='John Doe', email='john@example.com')
db.session.add(novo_usuario)
db.session.commit()

# Consultando usuários
usuarios = Usuario.query.all()
for usuario in usuarios:
    print(usuario.nome, usuario.email)

Outros Conceitos

O Flask oferece muitos outros recursos, como renderização de templates, manipulação de bancos de dados e uso de blueprints para organizar seu aplicativo de maneira modular. Explore a documentação do Flask para aprender mais!

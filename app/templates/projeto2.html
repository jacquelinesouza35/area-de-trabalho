Página de Projetos - Flask: Rotas e Inicialização de Formulários

Bem-vindo à página de projetos! Aqui vamos explorar como criar rotas e inicializar formulários dentro de um projeto Flask.

Criação de Rotas no Flask

No Flask, você pode criar rotas usando a decorator @app.route(). Isso permite definir o que será exibido quando um determinado URL for acessado.


from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return 'Página inicial do meu site.'

@app.route('/sobre')
def sobre():
    return 'Página de informações sobre o projeto.'

if __name__ == '__main__':
    app.run()

Inicialização de Formulários

Para trabalhar com formulários no Flask, você pode usar a extensão Flask-WTF. Isso permite criar, validar e processar formulários de maneira eficaz.


from flask import Flask, render_template
from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField

app = Flask(__name__)

class MeuFormulario(FlaskForm):
    nome = StringField('Nome')
    enviar = SubmitField('Enviar')

@app.route('/formulario', methods=['GET', 'POST'])
def formulario():
    form = MeuFormulario()
    if form.validate_on_submit():
        nome = form.nome.data
        return f'Nome submetido: {nome}'
    return render_template('formulario.html', form=form)

if __name__ == '__main__':
    app.run()

Outros Conceitos

O Flask oferece muitos outros recursos, como renderização de templates, manipulação de bancos de dados e uso de blueprints para organizar seu aplicativo de maneira modular. Explore a documentação do Flask para aprender mais!
